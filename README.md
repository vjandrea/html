Html extension
==============

This packages adds some missing methods to Illuminate/Html

Methods
-------

* favicon($href, $type)
* meta_charset($charset)
* html_lang($lang)
* title($title)
* h1($title, $properties)
* h2($title, $properties)
* h3($title, $properties)
* h4($title, $properties)
* h5($title, $properties)
* h6($title, $properties)
* heading($type, $title, $properties)
